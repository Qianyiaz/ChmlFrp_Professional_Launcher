# 发送邮箱验证码
## 请求参数
| 参数 | 描述 | 例举 |
| ------- | ------- | ------- |
| type | 验证码类型 | "register"为注册类型验证码, "retoken"为重置令牌验证码 |
| mail | 邮箱 | <2976779544@qq.com> |
## 成功API返回
```
{
    "msg": "发送成功",
    "code": 200,
    "state": "success"
}
```
## 失败API返回
```
{
    "msg": "邮箱格式错误",
    "code": 400,
    "state": "fail"
}
```
